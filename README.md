### Endpoints include:
* POST /api/organizers
* POST /api/venues
* POST /api/events
* POST /api/tickettypes
* GET /api/events
* GET /api/events/{id}
* POST /api/attendees
* POST /api/bookings
* PUT /api/bookings/{id}/cancel
* GET /api/events/{id}/revenue
* GET /api/attendees/{id}/bookings
